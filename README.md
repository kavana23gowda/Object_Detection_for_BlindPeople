# Object_Detection_for_BlindPeople
Real-time object detection system for visually impaired users that announces detected objects using YOLOv8 and text-to-speech.
Object Detection for Blind Users

Real-time object detection system designed to assist visually impaired users.
Uses YOLOv8 for detecting objects via webcam and announces them aloud using text-to-speech.
Configurable detection confidence, speech rate, and announcement gap make it responsive and user-friendly.

Features:
Detects multiple objects in real-time using a webcam.
Announces detected objects via text-to-speech.
Configurable confidence threshold and speech rate.
Avoids repeating announcements for objects already detected in the current frame.
